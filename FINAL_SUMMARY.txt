🚀 HOW TO USE
===============================================================================

LOCAL USE (Dark Mode - Default):
  $ pip install -r requirements.txt
  $ streamlit run app.py
  → Opens: http://localhost:8501
  → Dark mode enabled by default
  → Click ☀️ to switch to light mode

LOCAL USE (Light Mode):
  $ streamlit run app.py
  → Click ☀️ Light Mode button in left sidebar

SHARE WITH TEAM (Temporary URL):
  $ python start_network_tunnel.py
  → Get: https://abc-12345-xyz.ngrok.io
  → Share immediately
  → URL changes on next restart

SHARE WITH TEAM (Persistent URL):
  $ set NGROK_AUTHTOKEN=your_token_here
  $ python start_network_tunnel.py
  → Same URL every restart
  → Requires free ngrok account
  → Great for documentation

ONE-CLICK WINDOWS:
  → Double-click: START_NETWORK_SHARING.bat
  → Automatic setup and launch

🔐 SECURITY FEATURES
===============================================================================

✓ HTTPS Encryption
  - ngrok provides SSL/TLS encryption
  - All network traffic encrypted
  - No unencrypted local communication

✓ Automatic URL Rotation
  - URLs change each restart (without auth token)
  - Prevents unauthorized persistent access
  - Improves security posture

✓ Session Isolation
  - Each user gets isolated session
  - No cross-session data leakage
  - Browser isolation maintained

✓ Access Logging

  - Monitor who accessed when

🆘 SUPPORT & TROUBLESHOOTING
===============================================================================

COMMON ISSUES:

Q: Can't see theme buttons?
A: Scroll down in left sidebar. Check browser zoom level.

Q: Theme not switching?
A: Clear browser cache: Ctrl+Shift+Delete
   Try incognito/private mode

Q: No network URL generated?
A: Install pyngrok: pip install pyngrok
   Check internet connection
   Try: pip install --upgrade pyngrok

Q: URL keeps changing every restart?
A: Normal behavior. Get auth token for persistent URL:
   1. https://dashboard.ngrok.com
   2. set NGROK_AUTHTOKEN=token
   3. Restart

Q: Can't access network URL from team member?
A: Check firewall settings
   Verify ngrok status: status.ngrok.com
   Try from different network

FULL TROUBLESHOOTING:
  See: THEME_AND_NETWORK_GUIDE.md (troubleshooting section)
