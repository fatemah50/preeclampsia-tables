## ⚡ FASTEST OPTION (2 minutes)

### If you want to share your app TODAY:

1. **Open browser**: Go to https://share.streamlit.io
2. **Sign in**: Use your GitHub account
3. **New app**: Click "New app" button
4. **Select repo**: Choose your preeclampsia repo
5. **Select file**: Choose `app.py`
6. **Deploy**: Click Deploy button
7. **Share**: Copy the public URL and send to your team!

**That's it!** Your app is now live on the internet.

## 🛠️ LOCAL OPTION (Test First)

### If you want to test locally before sharing:

**Windows Users**:
```bash
double-click deploy_streamlit.bat
```

**macOS/Linux Users**:
```bash
bash deploy_streamlit.sh
```

**Then open**: http://localhost:8501

Test the app, make sure everything works, then deploy to Streamlit Cloud using the first option.
